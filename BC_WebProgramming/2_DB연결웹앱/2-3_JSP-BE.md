## 3. JSP - BE
JSP(Java Server Page)
JSP는 항상 Servlet으로 변환된 후 실행된다.

### 선언문(Declaratoin)
JSP내에 필요한 멤버변수나 메소드를 선언
```
<%!
// _jspinit()
code
%>
```
### 스크립트릿(Scriptlet)
```
<%
code
%>
```
### 표현식(Expression)
```
<%= %>
```
### 주석(Comment)
```
<!-- HTML주석 -->
<%-- JSP주석 --%>
// Java주석
/*
Java
주석
*/
```
