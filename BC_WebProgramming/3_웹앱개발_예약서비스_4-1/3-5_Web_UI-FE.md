#### Javascript

### DOMContentLoaded vs load

#### DOMContentLoaded

- DOMTree 분석이 끝나면 발생
- document 이벤트

#### load

- 모든 자원들이 다 받아져 브라우저에 렌더링까지 끝나면 발생
- window 이벤트
